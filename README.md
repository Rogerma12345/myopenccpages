# 文本转换与时间戳工具（纯静态页）

一个无需后端的单文件静态工具页（`index.html`），包含两类功能：

- 简体 → 繁体（台湾用语）：`s2twp`
- 繁体 → 简体（标准）：`t2s`
- Unix 时间戳 → 时间显示：支持秒/毫秒输入，并输出本地时间、UTC、ISO、相对时间（fromNow）

UI 采用常见静态工具页结构（`section + label + grid row + note/ok/err`），可直接部署到任意静态托管。

# 使用

1. 下载仓库（或仅复制 `index.html`）
2. 用浏览器直接打开 `index.html`  
   或部署到任意静态托管（如 GitHub Pages、Cloudflare Pages 等）

## 简繁转换说明

- `s2twp`：简体转台湾正体，并包含常见地域用词转换（例如“软件”→“軟體”）
- `t2s`：繁体转简体（标准）

如需更多配置（例如 `tw2sp`、`hk2s` 等），可在页面中新增按钮或改为下拉选择；这些配置名来自 OpenCC 官方配置集合，opencc-wasm 内建可用。

## 时间戳转换说明

输入支持：
- 13 位毫秒：例如 `1592483696789`
- 10 位秒：例如 `1592483696`

输出包含：
- 毫秒(ms)、秒(s)
- 本地时间（自动识别时区）
- UTC 时间
- ISO 8601
- 相对时间（fromNow）
